Product: superclass
    - int id
    - double price

Computer: subclass of Product
    - int width
    - int height

Dress: subclass of Product
    - String size (S, M, L, XL etc.)

Favorites: **generic** class, contains an instance variable of **generic List**.
    - addToFavorites(GenericType object) -> add this object to generic list of this class
